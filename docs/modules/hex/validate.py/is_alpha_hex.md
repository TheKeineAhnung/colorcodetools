# is_alpha_hex()

## How to use

1. ```python
   from colortools.hex.validate import is_alpha_hex
   is_alpha_hex('hexcode')
   ```
2. ```python
   import colortools
   colortools.hex.validate.is_alpha_hex('hexcode')
   ```

## Parameters

1. Required parameters:

   - hexcode:  
      Enter hex code which should be checked. The hex code must have alpha values
     - `type: string`
     - `default: None`

## Return value

- `type: boolean`
