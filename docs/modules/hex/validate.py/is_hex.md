# is_hex()

## How to use

1. ```python
   from colortools.hex.validate import is_hex
   is_hex('hexcode')
   ```
2. ```python
   import colortools
   colortools.hex.validate.is_hex('hexcode')
   ```

## Parameters

1. Required parameters:

   - hexcode:  
      Enter hex code which should be checked.
     - `type: string`
     - `default: None`

## Return value

- `type: boolean`
