# is_single_cymk()

## How to use

1. ```python
   from colorcodetools.cymk.validate import is_single_cymk
   is_single_cymk(c/y/m/k)
   ```
2. ```python
   import colorcodetools
   colorcodetools.cymk.validate.is_single_cymk(c/y/m/k)
   ```

## Parameters

1. Required parameters:

   - cymk:  
      Enter a single cymk code part.
     - `type: int`
     - `default: None`
     - `example: 25`

## Return value

- `type: boolean`
