# invert_cymk()

## How to use

1. ```python
   from colorcodetools.rgb.invert import invert_rgb
   invert_cymk('c/y/m/k')
   ```
2. ```python
   import colorcodetools
   colorcodetools.cymk.invert.invert_cymk('c/y/m/k')
   ```

## Parameters

1. Required parameters:

   - cymk:  
      Enter cymk code which should be inverted.
     - `type: string`
     - `default: None`
     - `example: '100/5/67/0'`

## Return value

- `type: boolean or string`
